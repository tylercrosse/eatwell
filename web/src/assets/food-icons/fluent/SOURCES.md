# Fluent food icon assets

These SVGs are vendored from Microsoft Fluent Emoji Flat for runtime food, meal,
and small app UI icons.

- Source repository: https://github.com/microsoft/fluentui-emoji
- License: MIT, https://raw.githubusercontent.com/microsoft/fluentui-emoji/main/LICENSE
- Style: Flat SVGs from `assets/<Emoji name>/Flat/`; skin-tone-aware emoji may
  come from `assets/<Emoji name>/Default/Flat/`

The category-to-asset map lives in `web/src/lib/foodCategoryIcons.tsx`; meal
icons live in `web/src/lib/mealIcons.ts`; app UI icons live in
`web/src/lib/appIcons.ts`.
